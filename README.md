# Game Test Automation
Automated testing for gaming services like Steam API, Epic Games Store, and game launchers.

## Tech Stack
- Python 3
- Pytest
- Selenium WebDriver (for UI tests)
- Requests (for API testing)
- Allure Report (for test reporting)
